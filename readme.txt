Расчет ливневой канализационной сети.

Логунцов С.В., Логунцов В.Ф. 2002 г.

Программа позволяет выполнить расчет ливневой канализационной сети с использованием методов численного моделирования.

Возможности программы

1.  Расчет дождевой канализационной сети с автоматическим выбором диаметров,
    уклонов, и автоматическим расчетом отметок.
2.  Программа обеспечивает минимизацию земляных работ на участках сети.
3.  Возможность изменения сортамента труб.
4.  На участке могут быть приняты перепадные колодцы, тем самым достигается
    минимизация земляных работ.
5.  Программа выполнена с учетом всех требований СНиП
